## ビルドに必要なアプリケーション

- [Grunt: The JavaScript Task Runner](http://gruntjs.com/)
- [Bower - A package manager for the web](http://bower.io/)

## モジュールのインストール

```
cd プロジェクトルート(bower.json, package.jsonが存在する場所)
npm install
bower install
```

## ファイルの変更を監視して継続的ビルド

```
grunt start
```

## 一度だけビルド

```
grunt build
```
